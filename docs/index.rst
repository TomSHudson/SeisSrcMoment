SeisSrcMoment Documentation
============

:mod:`SeisSrcMoment` is a Python package for earthquake moment and magnitude analysis.  

SeisSrcMoment provides methods for calculating the seismic momment release of an earthquake using time-domain and spectral methods. It also allows for bulk analysis of entire catalogues and post-processing such as finding b-values and temporal b-value variation through time.

The source code for the project is hosted on |github|.
